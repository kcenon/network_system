Detailed Description

Represents a WebSocket connection to a client.

This class provides an interface for interacting with individual WebSocket connections. It implements the i_websocket_session interface for composition-based usage.

Thread Safety:

  • All public methods are thread-safe
  • Can be safely shared across threads

Interface Compliance

This class implements interfaces::i_websocket_session.
