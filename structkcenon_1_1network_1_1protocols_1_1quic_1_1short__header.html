Detailed Description

QUIC Short Header format (RFC 9000 Section 17.3)

Short headers are used after the handshake completes. They are more compact and include only the destination connection ID.

Format: +-+-+-+-+-+-+-+-+ |0|1|S|R|R|K|P P| First byte: Header Form (0), Fixed Bit (1), +-+-+-+-+-+-+-+-+ Spin Bit, Reserved, Key Phase, PN Length |Destination Connection ID (0..160)| +-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+ |Packet Number (8..32) | +-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+
