Detailed Description

Adapter that wraps QUIC connection to implement i_connection.

This adapter bridges the QUIC protocol implementation with the unified interface, enabling protocol factory functions to return i_connection while using the QUIC connection underneath.

QUIC Specifics

  • Uses Stream 0 (bidirectional) for default data transfer
  • Supports multiplexed streams internally
  • Built-in TLS 1.3 encryption
  • Connection migration support

Thread Safety

Thread-safe. All methods can be called from any thread.

Ownership

The adapter owns the underlying QUIC connection and manages its lifecycle.
